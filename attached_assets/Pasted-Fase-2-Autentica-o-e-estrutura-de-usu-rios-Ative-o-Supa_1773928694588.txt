Fase 2 — Autenticação e estrutura de usuários

"Ative o Supabase e implemente a autenticação real. Crie as três roles (admin, gerente, cliente) com Row Level Security. Implemente o fluxo de primeiro login do admin com troca obrigatória de senha. Configure o redirecionamento pós-login para o painel correto de cada tipo de usuário. Nenhum dado mock — tudo real."

Fase 3 — Gestão de projetos (admin e gerente)

"Implemente a criação, edição, duplicação e exclusão de projetos com todas as informações básicas do formulário. Projetos salvos no Supabase em tempo real. Lixeira com exclusão automática após 15 dias via pg_cron. Histórico de alterações de status. Performance: select apenas campos necessários, sem select *, paginação na listagem."

Fase 4 — Gestão de clientes e gerentes

"Implemente o cadastro de clientes e gerentes usando a API Admin do Supabase via Edge Function (inviteUserByEmail — nunca signUp direto do frontend). Envio de e-mail via Resend com link para criação de senha. Desativação, exclusão e redefinição de senha. Correção do bug de criação de cliente."

Fase 5 — Cronograma

"Implemente a criação e edição do cronograma dentro de cada projeto: tabela editável inline, etapas padrão, drag-and-drop para reordenar, cálculo automático de status pelo fuso America/Sao_Paulo, exportação Excel com SheetJS. Visualização do cliente com timeline vertical, tabela alternável e badge 'Você está aqui'."

Fase 6 — Monitoramento de campo

"Implemente a configuração de cotas (4 tipos: mapeamento numérico, faixas etárias, texto/categoria, booleana) e a integração com Google Sheets via Edge Function com pg_cron a cada 5 minutos — o frontend nunca chama a planilha diretamente. Entrada manual como alternativa. Visualização do cliente com barras de progresso e gráficos Recharts."

Fase 7 — Portal do cliente (página de projeto)

"Implemente a página detalhada do projeto na visão do cliente: Visão Geral, Cronograma, Monitoramento de Campo, Documentos (upload no Supabase Storage com URL assinada), Histórico, Contato e Suporte. Badge 'Novo' quando houver atualização. Termômetro de Saúde automático."

Fase 8 — Financeiro

"Implemente a aba Financeiro: painel global (admin), painel do gerente (apenas seus projetos), detalhes financeiros por projeto com custos operacionais, distribuição de lucro protegida por senha do módulo (apenas admin), previsão de receita e calculadora de precificação com as fórmulas completas."

Fase 9 — KPIs, Metas, Alertas e Notificações

"Implemente o Dashboard de KPIs (somente admin), Metas mensais/trimestrais, sistema de Alertas de projetos em risco e Central de Notificações com sino no navbar para todos os tipos de usuário."

Fase 10 — Recursos finais

"Implemente: NPS automático ao encerrar projeto (e-mail via Resend + formulário público), página 'Sobre a Clarifyse' no portal do cliente, Log de Atividades do sistema, Parceiros de Painel, Configurações completas do administrador, todas as exportações em PDF e Excel.
